## 📖 Project Description

**FitZone Gym Website** is a modern and fully responsive fitness website developed using **HTML, CSS, and JavaScript**. The project is designed to provide an engaging online platform for gyms and fitness centers, allowing users to explore gym services, membership plans, trainers, testimonials, and contact information.

The website features a clean and attractive user interface with multiple interactive sections, including a **BMI Calculator**, **Membership Plans**, **Trainer Profiles**, **Customer Testimonials**, **Contact Form**, **Google Maps Integration**, and a **WhatsApp support button**. The responsive design ensures an optimal viewing experience across desktops, tablets, and mobile devices.

This project demonstrates front-end web development skills such as responsive layouts, modern UI design, JavaScript DOM manipulation, form handling, and user interaction features. It can serve as a portfolio project for aspiring web developers or as a template for fitness businesses seeking an online presence. 

---

### ✨ Features

* Responsive and modern UI design
* Hero section with call-to-action buttons
* About and Services sections
* Membership pricing plans
* Trainer profiles
* Customer testimonials
* BMI Calculator using JavaScript
* Contact form validation
* Google Maps integration
* WhatsApp floating contact button
* Mobile-friendly design
